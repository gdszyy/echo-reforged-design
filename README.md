# Echo Reforged - 游戏设计文档库

> **创建日期**: 2025-01-06

## 项目概述

《Echo Reforged》是一款融合**回合制打砖块战斗**、**动态世界探索**与**深度 Roguelike Build 构建**的游戏。本仓库用于存储和迭代游戏的核心设计文档。

### 前置项目

本项目的设计理念源自两个概念验证 MVP：

- **Echo Alchemist**: 验证弹珠/打砖块战斗机制与属性系统
- **World Morphing Simulator**: 验证动态世界演化与细胞自动机系统

## 核心设计支柱

1. **复合 Build 系统**: 钉盘养成 + 子弹编程 + 符文填字谜
2. **世界即战场**: 战斗场地是世界场景的一部分，地块属性影响战斗
3. **玩家影响世界**: 玩家的行为（如开采资源）会触发世界变迁的连锁反应

## 设计原则

- **有意义的选择**: 每个决策都应带来权衡
- **涌现式玩法**: 系统间的交互产生意想不到的策略
- **世界的真实感**: 玩家的行为有逻辑性的后果
