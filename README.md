## 项目简介

《Echo Reforged》是一款融合了 **Roguelike 元素**、**回合制弹珠战斗** 和 **深度 Build 构建** 的独立游戏。玩家将扮演一位在无尽轮回中觉醒的旅者，在一个由晶石与以太构成的动态世界中探索、战斗、成长，并最终追溯自己的神话。

### 前置项目

本项目的设计理念源自两个概念验证 MVP：

- **Echo Alchemist**: 验证弹珠/打砖块战斗机制与属性系统
- **World Morphing Simulator**: 验证动态世界演化与细胞自动机系统

### 核心玩法支柱

| 支柱 | 描述 |
| :--- | :--- |
| **战斗** | 回合制"打砖块"战斗，融合物理弹射的爽快感与策略构建的深度。 |
| **探索** | 在一个会动态演化的世界中探索，玩家的行为会影响世界的走向。 |
| **构建** | 通过"钉盘养成"、"子弹编程"和"符文填字谜"三大系统，打造独一无二的 Build。 |

## 设计亮点

1. **"波粒二象性"战斗哲学**: 前期"炼金战斗"强调随机物理模拟，后期"修仙战斗"则转变为可控的规则计算，统一在同一个世界观框架下。

2. **"神话叙事"系统**: 玩家的行为会被记录并转化为下一周期世界中的传说，最终追溯自我神话，形成宏大的叙事闭环。

3. **"晶石博弈"**: β晶石既是核心资源，也是抑制危险α晶石的守护者。开采资源会打破世界平衡，迫使玩家做出有意义的策略抉择。

4. **"世界即战场"**: 战斗场景由世界地块属性动态生成，探索与战斗深度绑定。

---

## 设计原则

- **有意义的选择**: 每个决策都应带来权衡
- **涌现式玩法**: 系统间的交互产生意想不到的策略
- **世界的真实感**: 玩家的行为有逻辑性的后果
