# Echo Reforged - 炼金-修仙体系融合设计文档

> **文档版本**: 1.0
> **最后更新**: 2025-01-06

本文档旨在详细阐述《Echo Reforged》中，如何将游戏前期的“西方炼金”风格平滑地过渡到后期的“东方修仙”体系，并确保在世界观、游戏机制和战斗系统上的内在统一性。

---

## 1. 核心概念映射：从炼金到修仙

两个体系的底层逻辑是统一的，区别仅在于不同时代、不同认知水平的人类对其的称呼和理解方式不同。

| 核心概念 | 西方炼金（蛮荒时代） | 东方修仙（觉醒时代后） | 底层逻辑 |
| :--- | :--- | :--- | :--- |
| **能量来源** | 以太 (Aether) | 灵气 / 天地元气 | **同一事物，不同称呼**。 |
| **能量载体** | 晶石 (Crystal) | 灵石 | **同一事物**。β晶石即为灵石，α晶石即为“魔晶”或“邪灵石”。 |
| **能量转化工具** | 发射器 + 钉板 + 弹珠 | 法宝 + 法阵 + 法力 | **同一事物的不同发展阶段**。发射器是“未觉醒的法宝”。 |
| **材料加工** | 炼金术 (Alchemy) | 炼丹术 / 炼器术 | **同一技术的不同分支**。 |
| **个人强化** | 晶化（被动/危险） | 修仙（主动/可控） | **同一过程的不同掌控程度**。 |
| **知识传承** | 古代石板解谜 / 蓝图 | 功法 / 秘籍 | **同一事物的不同形态**。 |

**设计核心**：游戏通过NPC对话和环境叙事，让玩家自然地经历这个术语和认知的演变，而非生硬地切换系统。

---

## 2. 世界演变：里程碑触发机制

世界的演变由玩家达成的特定“里程碑”触发，确保了叙事的逻辑性和玩家的主导性。

| 里程碑 | 触发条件 | 世界变化 | 解锁内容 |
| :--- | :--- | :--- | :--- |
| **文明火种** | 玩家成功将一个部落发展为**城市**。 | 世界进入**稳定期**，人类聚落增加，出现贸易。 | 解锁“城市”经营内容。 |
| **第一代新生儿** | 城市中诞生了第一批**自然繁衍**的人类后代。 | 世界时间开始加速流逝。 | 解锁“人口世代”系统。 |
| **灵根觉醒** | 经过数代繁衍后，第一个拥有**灵根**的“新人类”诞生。 | 世界进入**觉醒时代**，出现野生修仙者。 | 解锁玩家的“修仙”系统（练气期）。 |
| **宗门雏形** | 修仙者数量达到一定规模。 | 世界进入**宗门时代**，势力开始形成。 | 解锁“宗门”系统（声望、任务）。 |
| **仙凡分离** | 出现第一个达到**炼虚期**或更高境界的修仙者。 | 世界进入**仙凡分离时代**，高阶修仙者脱离凡俗。 | 解锁“洞天福地”系统，进入终局剧情。 |
| **追溯真相** | 玩家收集到所有关于“最初神话”的线索。 | 触发**终局剧情**。 | 揭示世界真相，做出最终选择。 |

## 3. 战斗系统的平滑扩展

修仙能力是对现有战斗系统的**叠加和扩展**，而非替代。

| 境界 | 战斗系统变化 | 具体机制 |
| :--- | :--- | :--- |
| **凡人** | **纯发射器战斗** | 游戏的核心玩法基础。 |
| **练气期** | **+以太感知** | 解锁“以太视野”看穿敌人弱点；解锁“以太灌注”为弹珠附加基础属性。 |
| **筑基期** | **+晶核蓄能** | 解锁“晶核”系统，积攒能量释放“晶核爆发”（大招）。 |
| **金丹期** | **+子弹预设** | 消耗丹力，**强制指定**弹珠的一个或多个属性，减少随机性。 |
| **元婴期** | **+元婴分身** | 释放元婴进行**独立、可控的攻击**，相当于一个“副武器”。 |
| **炼虚期** | **+法宝觉醒 / 子弹编程** | 发射器觉醒为本命法宝，解锁“子弹编程”完全体，可精确控制弹珠路径和效果。 |
| **合道期** | **+意念凝形** | 可选择完全放弃发射器，直接以意念凝聚以太进行攻击（通关后的“无双模式”）。 |

## 4. 生存机制的演变

从“一触即死”到“HP系统”的平滑过渡，以适应不同阶段的敌人和战斗模式。

| 阶段 | 生存机制 | 世界观解释 |
| :--- | :--- | :--- |
| **凡人** | **一触即死** | 凡人肉体不堪一击。 |
| **练气期** | **+护体灵气 (一次容错)** | 学会用灵气在体表形成保护罩，可抵挡一次致命伤害。 |
| **筑基期** | **+护盾值** | 晶核反哺肉身，形成可恢复的护盾条。护盾破后，仍会触发“一触即死”或“钉板损坏”。 |
| **金丹期** | **+护盾强化** | 护盾容量提升，并可自动恢复。 |
| **元婴期** | **+元婴替死** | 元婴可在关键时刻替主人承受致命一击。 |
| **炼虚期及以上** | **+传统HP系统** | 肉身完成晶化，对普通攻击免疫，拥有真正的生命值条。 |

**敌人攻击类型**：为了丰富战斗策略，敌人的攻击被分为物理攻击、法术攻击、破盾攻击、真伤攻击等，对玩家不同的防御层有不同的效果。
