# 叙事、经营与遗产系统设计文档

> **文档版本**: 1.0
> **最后更新**: 2025-01-06

## 1. 概述

本篇文档旨在为《Echo Reforged》构建一个富有深度和情感连接的宏大叙事框架。它由三个紧密相连的系统构成：**部落经营系统**（玩家与世界交互的核心）、**跨周期遗产系统**（Roguelike循环中的永久成长）和**神话叙事系统**（将玩家行为转化为世界历史）。这套体系将游戏从一个玩法驱动的模拟器，升华为一个以“自我追溯”为主题的史诗故事。

---

## 2. α晶石“辐射”的重新定义

在深入设计之前，我们首先将α晶石的“辐射”概念进行升华：

- **本质**: 它不再是简单的物理伤害，而是晶石生态对生物圈的一种**抽象化的、持续性的生存压力**。它代表了晶石文明的扩张对原生文明的“挤压”和“侵蚀”。
- **体现**: 在游戏中，它表现为对生物聚落**繁荣度 (`prosperity`)** 的持续削减。繁荣度成为了部落抵御这场“天灾”的生命线。
- **意义**: 这一设计将玩家的行为（开采β晶石导致α晶石爆发）与整个世界的生态平衡和文明存续联系在了一起，赋予了玩家决策更宏大的背景和更沉重的后果。

---

## 3. 部落经营系统：从盟友到“永恒之城”

该系统旨在让玩家与世界建立深度的情感连接，并通过自己的努力留下永恒的印记。

### 3.1 信任度 (Trust)

- **核心机制**: 每个友好部落都拥有一个对玩家的**信任度**数值（0-100）。
- **获取方式**: 玩家可以通过以下行为提升信任度：
    - **守城援助**: 成功帮助部落抵御一次怪物潮或敌对部落的进攻。
    - **资源捐赠**: 向部落捐赠其急需的资源（如β晶石、特殊材料）。
    - **解决威胁**: 清除部落周围一个强大的敌人巢穴或危险的α晶石区域。
    - **完成委托**: 完成部落发布的随机任务（如寻找失踪的村民、护送商队）。

### 3.2 发展决策权 (Governance)

- **解锁**: 当信任度达到特定阈值（如 80），玩家将获得该部落的**发展决策权**。
- **经营玩法**: 玩家可以在部落的“议事厅”中，消耗资源和时间来推行各种发展策略，将部落从一个原始聚落逐步发展为一座坚固的城市。

| 发展方向 | 具体决策项 | 效果 |
| :--- | :--- | :--- |
| **军事** | 训练守卫、建造箭塔、加固城墙 | 提升部落在世界变迁中的自卫能力，降低被怪物潮摧毁的概率。 |
| **商业** | 建立市场、吸引商队、开发特产 | 提升部落商店的商品等级和稀有度，玩家可以买到更好的发射器插件和消耗品。 |
| **科技** | 建立工坊、研究晶石科技、设立图书馆 | 解锁独特的发射器、钉子蓝图，或提升部落对α晶石辐射的抗性。 |
| **文化** | 修建神庙、举办庆典、记录历史 | 提升部落的繁荣度上限，并开始**记录和传颂玩家的事迹**（为神话叙事系统提供素材）。 |

### 3.3 “永恒之城” (The Eternal City)

- **跨周期遗产**: 当一个部落被玩家发展为“城市”（所有发展方向均达到一定等级）后，它将成为一个**跨周期存续的遗产**。
- **意义**: 即使玩家在该周期中死亡，这座由他亲手建立的城市及其所有功能（高级商店、科技蓝图、历史记录）都会**完整地保留到下一个周期**。它成为了玩家在无尽轮回中唯一的家园和永恒的见证。

## 4. 神话叙事系统：我即是传说

该系统是游戏的核心情感驱动力，旨在创造一种“蓦然回首，自己已成神话”的独特体验。

### 4.1 行为记录与传说生成

- **触发器**: 当玩家在一个周期中完成某些“壮举”时，系统会自动记录这些事件。
- **壮举示例**:
    - 单枪匹马摧毁一个强大的怪物巢穴。
    - 在一次战斗中造成超高额的伤害。
    - 成功将一个部落发展为城市。
    - 在一次探索中深入世界的未知区域。
    - 大量开采β晶石导致α晶石爆发，间接摧毁了一个部落（负面壮举）。
- **传说生成**: 当玩家死亡并开启新周期后，系统会基于上个周期的记录，在世界中随机生成相关的**传说片段**。这些片段会通过以下形式呈现给新周期的玩家：
    - **吟游诗人的歌谣**: 在城市的酒馆里，吟游诗人弹唱着“独战千军的英雄”或“引来天灾的魔王”的史诗。
    - **残破的石碑**: 在野外的废墟中，玩家可能发现一块石碑，上面刻着“一位无名旅者曾在此地斩杀巨兽”。
    - **村民的闲谈**: 部落里的老人会向孩子们讲述“那位带来繁荣的先祖”的故事。

### 4.2 “自我追溯”的叙事循环

1.  **初闻传说**: 新周期的玩家在探索世界时，会不断听到、看到关于前人的传说。
2.  **发现关联**: 玩家会慢慢发现，这些传说中描述的地点、事件甚至使用的武器，都与自己上个周期的经历惊人地相似。
3.  **恍然大悟**: 在某个关键节点（例如，找到了上个周期自己死亡时掉落的、已成为“传说遗物”的发射器），玩家会恍然大悟——**那些传说的主角，就是过去的自己**。

### 4.3 终极驱动力：追溯最初的神话

- **初始神话**: 游戏开篇会通过一段动画或文本，向玩家讲述一个宏大的、关于世界起源、晶石来源以及一位“初代英雄”的**初始神话**。
- **最终目标**: 当玩家经历了数次“自我追溯”的循环，并意识到自己有能力将行为刻入世界历史后，他的最终目标将转变为——**追溯并验证那个“初始神话”的真相**。
- **终极揭示**: 游戏的最终结局，将是玩家通过自己的努力，一步步揭开“初始神话”的秘密，并最终发现，那个被传颂了无数个世代的“初代英雄”，或许……也是自己。这形成了一个完美、宏大且充满宿命感的叙事闭环。

---

通过这套系统，《Echo Reforged》将不仅仅是一款好玩的 Roguelike 游戏，更将成为一部由玩家亲手书写、并在轮回中亲自阅读的个人史诗。
