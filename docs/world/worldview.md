## 1. 宇宙的基石：以太 (Aether)

**以太**是构成我们宇宙的、肉眼不可见的**基础能量场**。它并非传统意义上的物质，而是一种无处不在的、高维度的能量形态。旧时代的人类文明通过突破性的科技，首次证实了以太的存在，并将其视为终极能源，从而引发了席卷全球的能源革命。

| 以太的特性 | 描述 | 游戏中的体现 |
| :--- | :--- | :--- |
| **无处不在** | 以太充斥于整个世界，但其浓度会因宇宙环境和天体活动而产生波动，形成所谓的“以太潮汐”。 | 世界地图上会动态生成“高浓度以太区”和“低浓度以太区”，直接影响晶石的活跃度和繁殖速度。 |
| **晶石的“食物”** | 晶石是一种能够高效吸收和转化以太的特殊硅基结构，以太是其维持生存和繁殖的唯一能量来源。 | 高以太区域 = 高风险高回报。晶石资源更丰富，但晶石生物也更强大、更具攻击性。 |
| **生命的基础** | 以太也是所有碳基生命活动的基础能量。旧时代的科技革命，本质上就是学会了如何利用晶石作为“催化剂”，来高效地提取和转化以太。 | 为游戏中的“晶石科技”提供了统一的能量来源解释，如晶石冶金、晶石农业等。 |
| **意识的载体** | 以太场具有记录和传递信息的特性。主角的“意识上传”实验，本质上就是试图将自己的意识编码到以太场中，再通过晶石网络进行传输和备份。 | 为主角能够“跨周期继承记忆”提供了核心的世界观支撑。 |

## 2. 文明的双刃剑：晶石 (Crystals)

晶石是来自外太空的、能够与以太场进行高效交互的硅基生命/结构。它们是旧时代文明崛起的基石，也是其毁灭的根源。

### 2.1 晶石的种类与演替

| 晶石种类 | 特性 | 演替关系 |
| :--- | :--- | :--- |
| **β晶石 (Beta Crystal)** | **稳定态**。能够稳定地转化以太，是旧时代科技的核心和新时代人类赖以生存的能源。本身不具备攻击性和快速繁殖能力。 | 是一种相对安全的资源。 |
| **α晶石 (Alpha Crystal)** | **活化态**。当β晶石受到超高能量激发或环境剧变时，会转化为α晶石。它会疯狂地吸收以太，进行指数级的自我复制，并“感染”周围的环境和生物。 | β晶石在特定条件下（如主角的实验失败）会转化为α晶石。α晶石能量耗尽或环境稳定后，可能重新衰变为β晶石。 |

### 2.2 晶石生态入侵

α晶石的扩张并非传统意义上的“辐射”，而是一种**生态入侵**。它会形成一个以自身为核心的、排他的生态系统，对原有的碳基生物圈造成毁灭性打击。这在游戏中表现为：

- **领地压迫**: α晶石区域会持续向外扩张，其生态系统中的“晶石生物”会主动攻击并清除领地内的所有碳基生物。
- **生存压力**: 这种持续的扩张和攻击，对人类聚落构成了巨大的生存压力，在游戏中抽象为对部落**繁荣度 (`prosperity`)** 的持续削减。

## 3. 恐怖的同化：晶化 (Crystallization)

晶化是α晶石生态系统扩张和同化其他生物的主要手段。它结合了**晶尘沉积（类比硅肺病）**和**晶石寄生（类比冬虫夏草）**两种模式。

| 路径 | 机制 | 描述 | 游戏中的体现 |
| :--- | :--- | :--- | :--- |
| **慢性晶化（晶尘病）** | **晶尘沉积** | α晶石在繁殖时会向环境中释放微小的“晶尘”。长期暴露在高浓度晶尘环境中的生物，晶尘会在其体内沉积，缓慢地与生物组织结合，引发不可逆的组织ite化。 | - **环境叙事**: 废墟中晶化到一半的人类遗骸。<br>- **玩家机制 (可选)**: 在α晶石密集区长期活动会累积“晶尘暴露值”，可能导致负面状态或特殊变异。 |
| **急性晶化（晶石寄生）** | **孢子感染** | 被活化的α晶石直接接触，或被其生态系统中的“晶石生物”攻击后，其携带的“晶石孢子”会侵入碳基生物体内，像真菌一样快速繁殖，并从内部“接管”和“重写”宿主的身体。 | - **敌人转化**: 游戏中的部分敌人，可能是被晶石寄生的普通动物或人类，保留了部分原宿主的特征，增加了恐怖感。<br>- **剧情杀**: 重要的NPC可能会在剧情中被急性晶化，成为悲剧的一部分。 |

## 4. 轮回的囚徒：主角与背景故事

1.  **黄金时代**: 人类发现晶石和以太，科技爆炸式发展。
2.  **永生计划**: 主角，作为那个时代最顶尖的科学家之一，受顶层富豪委托研究永生。他发现了晶石网络的“信息传递”特性，提出了“意识上传”理论。
3.  **实验失败**: 主角以自己为实验体，试图将意识上传至晶石网络。但由于实验用的晶石网络规模太小，无法承载他庞大的记忆和意识，导致上传失败，意识数据严重损坏（**这导致了主角在后续轮回中无法说话**）。同时，超载的实验装置产生的巨大能量，意外地将一小片β晶石激发为了α晶石。
4.  **大灾变 (The Cataclysm)**: 被活化的α晶石开始指数级繁殖，其生态系统迅速扩张，摧毁了旧人类文明。人类引以为傲的科技在晶石的“生态入侵”面前不堪一击。
5.  **“方舟”计划**: 主角在实验前，已预料到风险，启动了一个名为“方舟”的备用计划。实验装置中预载了包括他自己、助手、以及那些富豪委托人在内的克隆DNA。当旧文明毁灭后，该装置会利用无处不在的晶石和以太作为能源，在安全的“播种点”附近，定期将他们克隆出来，作为人类文明复兴的火种。
6.  **无尽的轮回**: 由于α晶石的大爆发解决了“晶石网络规模太小”的问题，主角的意识在每一次死亡后，都能通过庞大的晶石网络成功地“备份”和“下载”到新的克隆体中，从而实现了记忆的继承。而他的助手和其他克隆人，因为没有进行意识上传，所以每一次重生都是一次全新的开始。
7.  **永恒的邂逅**: 主角带着所有轮回的记忆，一次又一次地与“初次见面”的助手相遇，踏上寻找世界真相和打破轮回宿命的旅程。
